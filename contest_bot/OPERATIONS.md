## What the artifact log records

Each row is `{ts, event, decision_id, payload}`. Events:

| Event | When |
|---|---|
| `gate_allow` | Gecko returned `act` with confidence ≥ 0.6 |
| `gate_block` | Gecko returned anything else; trade NOT taken |
| `gate_error` | Oracle 402/timeout/parse-fail; treated as block |
| `breaker_trip` | Hourly $3 drawdown pause fired; new entries blocked |
| `position_open` | An entry actually fired (post-gate-allow) |
| `position_close` | Position closed via TP / SL / time-stop |

The pair `gate_allow` + `position_open` + `position_close` forms one realized trade. The pair `gate_block` + nothing forms a *counterfactual* — what the ungated baseline would have done. **This is the BPAR comparison falling out for free.**
